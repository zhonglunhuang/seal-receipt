# 定金收據產生器

海豹生活股份有限公司 — 純前端定金收據產生工具。

選擇房源、輸入承租方姓名與定金金額，即可產生並下載一張收據圖片（PNG），直接傳給房客。

## 特點

- **純前端、零相依**：直接用瀏覽器開啟 `index.html` 即可使用，手機也能用。
- **可下載 / 複製**：以高解析度（1200×1500）輸出 PNG，檔名自動帶入承租方姓名與收據編號。
- **內建房源清單**：公館、後山埤、松山、市政府、昆陽、松隆、中坡南（另可自訂）。
- **不退款與押金條款**直接印在收據上：
  1. 定金一經收取，恕不退還。
  2. 入住後定金全額轉作押金，押金高於定金時於入住當日補足差額；押金於租約期滿、結清且無毀損時依約退還。
- 收款方固定為「海豹生活股份有限公司」，標註系統產生、毋需簽章。

## 線上使用

GitHub Pages：開啟本 repo 的 Pages 網址即可。

## 本機使用

直接雙擊 `index.html`，或：

```bash
open index.html
```
